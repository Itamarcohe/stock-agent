## איפה עצרנו

הסוכן הבא שצריך לבנות: **Dip Opportunity Agent**
קובץ: `app/agents/dip_agent.py`

הסוכן בודק האם ירידה חדה במניה היא הזדמנות קנייה או סכנה, על בסיס:
- כמה המניה ירדה מהשיא
- RSI
- מיקום מול MA20
- נפח מסחר

אחריו: Valuation Agent, Market Regime Agent, Long-Term Entry Agent ועוד.

---

## דרישות מערכת

- Python 3.10–3.13
- Node.js LTS
- חיבור אינטרנט (לשליפת נתונים מ-Yahoo Finance)
- Windows 10/11
